# Orbit Remote — Windows Desktop Client

Десктоп-клиент Orbit Remote (Electron) для управления Android-устройствами через
интернет. Подключается к рабочему сигнальному серверу, принимает экран телефона по
WebRTC и отправляет ввод (мышь/клавиатура) обратно через data channel.

## Возможности

- Подключение по **Device ID + код** к развёрнутому сигнальному серверу.
- Приём экрана телефона по **WebRTC** (видео), низкая задержка.
- Управление: **мышь → тап/свайп/удержание**, двойной клик → двойной тап,
  **колесо → прокрутка**, **клавиатура → ввод текста**, Esc → «Назад», Enter.
- Кнопки навигации: Назад, Домой, Recents, шторка уведомлений.
- Синхронизация буфера обмена (Ctrl+C / Ctrl+V).
- **Тёмная и светлая темы**, список и **избранные** устройства, поиск.
- **Мультисессии** — до 20 устройств одновременно (вкладки).
- Полноэкранный режим, скриншот экрана телефона (сохранение в PNG).


## Запуск (разработка)

```
cd windows
npm install
npm start
```

## Сборка инсталлятора Windows

```
npm run dist      # → dist/OrbitRemote-Setup.exe (NSIS)
```

Готовый `OrbitRemote-Setup.exe` можно положить в `releases/` сайта — кнопка скачивания
активируется автоматически.

## Использование

1. На телефоне открыть Orbit Remote (Android-агент) — он покажет **Device ID** и **код**.
2. В десктоп-клиенте ввести ID и код → **Connect** (опц. «Save device»).
3. Появится экран телефона. Управляйте мышью и клавиатурой.

Сменить адрес сигнального сервера: `preload.js` → `defaultSignalingUrl`.

## Примечания / ограничения

- WebRTC использует встроенный движок Chromium в Electron — нативных зависимостей нет.
- Ввод текста использует Accessibility `ACTION_SET_TEXT` на стороне Android; для полей
  со сложной логикой ввод посимвольно может вести себя неидеально — это ограничение API.
- Для управления телефон должен дать разрешения «Захват экрана» и «Спец. возможности».
- Сквозное шифрование обеспечивает WebRTC (DTLS-SRTP).
- Альтернатива стеку: renderer написан на чистом JS (без сборщика) для простоты сборки;
  при желании легко переносится на React/Vite.
